## Fitur Utama
Aplikasi ini memiliki fitur-fitur sesuai ketentuan tugas:

**1. Authentication**
* Login menggunakan Email & Password.
* Login menggunakan **Google Sign-In**.
* Register akun baru.
* Forgot Password & Change Password.

**2. Anime Browsing**
* Menampilkan daftar anime (Pagination / Load More trigger).
* **Pencarian (Search)** anime berdasarkan judul.
* **Filter** anime berdasarkan Genre.

**3. Favorites**
* Menambahkan anime ke daftar favorit.
* Menghapus anime dari daftar favorit.
* Data favorit tersimpan di Firestore secara realtime.

**4. User Profile**
* Menampilkan informasi pengguna.
* Logout.
